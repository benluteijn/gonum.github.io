Installing Gonum

  1. Follow the instructions to install Go
  2. Download a Gonum package using
    go get -u -t gonum.org/pkgname
    or
    go get -u -t gonum.org/pkgname/subpkg
  3. That's it!

For example, the matrix package implements functions and methods for using and manipulating matrices.
go get -u -t gonum.org/matrix

A simple "Hello, Gonum" script to test installation: package main import ( "fmt" "gonum.org/matrix" ) func main() { m := matrix.NewDense(5, 5, nil) fmt.Println("%v\n", matrix.Formatted(m)) }
